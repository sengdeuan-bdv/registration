ກັ່ນຕອງຂໍ້ມູນ (Filters)

ບົດສະຫຼຸບຜົນການວິເຄາະ (Pivot Table)

ຍິນດີຕ້ອນຮັບສູ່ລະບົບວິເຄາະຂໍ້ມູນ AppSheet

ກະລຸນາອັບໂຫຼດໄຟລ໌ CSV ທີ່ Export ມາຈາກ AppSheet ເພື່ອເລີ່ມຕົ້ນການສ້າງລາຍງານ.

ຄູ່ມືການນຳໃຊ້ລະບົບ (User Manual)

1. ການນຳເຂົ້າຂໍ້ມູນ (Upload Data)

ກົດປຸ່ມ "ອັບໂຫຼດ CSV" ຢູ່ແຖບດ້ານເທິງ ເພື່ອເລືອກໄຟລ໌ຂໍ້ມູນທີ່ທ່ານ Export ອອກມາຈາກ AppSheet. ລະບົບຈະທຳການອ່ານ ແລະ ສ້າງຕາຕະລາງສະຫຼຸບຜົນໃຫ້ອັດຕະໂນມັດ.

2. ການກັ່ນຕອງຂໍ້ມູນ (Filters)

  • ສາມາດກັ່ນຕອງໄດ້ຫຼາຍເງື່ອນໄຂ (Host, Domain, Type, Level, Date, Participant Type).
  • ສາມາດເລືອກໄດ້ຫຼາຍອັນພ້ອມກັນ (Multi-select) ໃນແຕ່ລະໝວດໝູ່.
  • ຫຼັງຈາກເລືອກແລ້ວ ໃຫ້ກົດປຸ່ມ "ຕົກລົງ (ປິດ)" ຢູ່ລຸ່ມສຸດຂອງໜ້າຕ່າງ Filter ນັ້ນ.
  • ກົດປຸ່ມ "ລ້າງການກອງທັງໝົດ" ຢູ່ມຸມຂວາເທິງຂອງແຜງກັ່ນຕອງ ເພື່ອຍົກເລີກເງື່ອນໄຂທັງໝົດ.

3. ຮູບແບບການຄິດໄລ່ (Count Type)

  • Count (ນັບທັງໝົດ): ລະບົບຈະນັບທຸກໆແຖວຂໍ້ມູນ (ເໝາະສຳລັບການນັບຈຳນວນຄັ້ງການເຂົ້າຮ່ວມ).
  • Distinct (ນັບບໍ່ຊ້ຳ ID): ລະບົບຈະນັບສະເພາະ Participant ID ທີ່ບໍ່ຊ້ຳກັນ (ປ້ອງກັນການນັບບຸກຄົນດຽວກັນຊ້ຳຊ້ອນກັນ).

4. ການເຊື່ອງຖັນ ແລະ ເຊື່ອງແຖວ

  • ເຊື່ອງຖັນທົ່ວໄປ: ກົດປຸ່ມຮູບຕາປິດ 👁️ ຢູ່ເທິງຫົວຂໍ້ຖັນ ຈະເປັນການເຊື່ອງຖັນນັ້ນຈາກການສະແດງຜົນ.
  • ເຊື່ອງຖັນ Organization Type: ສຳລັບຖັນທີ່ຢູ່ຫຼັງຈາກຖັນ "ລວມ" ໄປ, ເມື່ອກົດເຊື່ອງ ຂໍ້ມູນຂອງຄົນກຸ່ມນັ້ນຈະຖືກຕັດອອກຈາກການຄິດໄລ່ທັນທີ (ຈະບໍ່ຖືກນັບເຂົ້າໃນຜົນລວມ ຄືກັນກັບການເຊື່ອງແຖວ).
  • ເຊື່ອງແຖວ (Hide Row): ເອົາເມົ້າໄປຊີ້ໃສ່ຊື່ Event Name ໃນຕາຕະລາງ, ຈະມີໄອຄອນຕາປະກົດຂຶ້ນ. ເມື່ອກົດເຊື່ອງ ແຖວນັ້ນຈະຖືກເຊື່ອງ ແລະ ຕົວເລກຈະຖືກຕັດອອກຈາກການຄຳນວນຜົນລວມ.
  • ສາມາດກົດປຸ່ມ "ສະແດງທຸກຖັນ" ຫຼື "ສະແດງທຸກແຖວ" ທີ່ຢູ່ຂ້າງປຸ່ມ Export ເພື່ອຮຽກຂໍ້ມູນທີ່ເຊື່ອງໄວ້ກັບຄືນມາໄດ້ສະເໝີ.

5. ການສົ່ງອອກ (Export Excel)

ກົດປຸ່ມ "ສົ່ງອອກລາຍງານ Excel" ເພື່ອດາວໂຫຼດໄຟລ໌ .xlsx. ໄຟລ໌ Excel ຈະສະແດງສະເພາະຂໍ້ມູນທີ່ທ່ານກັ່ນຕອງ ແລະ ບໍ່ນຳເອົາຖັນ ຫຼື ແຖວທີ່ທ່ານເຊື່ອງໄວ້ອອກໄປນຳ, ພ້ອມທັງມີການສະແດງເງື່ອນໄຂ Filter ທີ່ໃຊ້ທັງໝົດໄວ້ເທິງຫົວຕາຕະລາງ.

1. Data Upload

Click the "Upload CSV" button at the top bar to select the CSV file exported from AppSheet. The system will read and generate the pivot table automatically.

2. Data Filtering

  • Filter by various criteria (Host, Domain, Type, Level, Date, Participant Type).
  • Multi-selection is supported for each category.
  • After selecting, click the "Apply (Close)" button at the bottom of the dropdown.
  • Click "Clear All Filters" at the top right of the panel to reset all conditions.

3. Calculation Methods (Count Type)

  • Count (Total): Calculates the total sum of all rows (useful for overall attendance count).
  • Distinct (Unique IDs): Counts only unique Participant ID values (prevents double-counting the same person).

4. Hide Columns & Rows

  • Hide General Columns: Click the 👁️ icon on the column header to hide it from view.
  • Hide Organization Type Columns: For dynamic columns after the "Total" column, hiding them will exclude their data from calculations immediately (reduces Grand Totals accordingly).
  • Hide Row: Hover over an Event Name to reveal the hide icon. Hiding a row removes it and excludes its values from the Grand Total.
  • Click "Show All Columns" or "Show All Rows" next to the Export button to restore hidden items anytime.

5. Excel Export

Click "Export Excel" to download a .xlsx file. The file replicates your filtered view exactly, omitting hidden rows/columns, and appends a header listing all currently active filter criteria.
