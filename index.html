ກັ່ນຕອງຂໍ້ມູນ (Filters)

ບົດສະຫຼຸບຜົນການວິເຄາະ (Pivot Table)

ຍິນດີຕ້ອນຮັບສູ່ລະບົບວິເຄາະຂໍ້ມູນ AppSheet

ກະລຸນາອັບໂຫຼດໄຟລ໌ CSV ທີ່ Export ມາຈາກ AppSheet ເພື່ອເລີ່ມຕົ້ນການສ້າງລາຍງານ.

ຄູ່ມືການນຳໃຊ້ລະບົບ (User Manual)

1. ການນຳເຂົ້າຂໍ້ມູນ (Upload Data)

ກົດປຸ່ມ "ອັບໂຫຼດ CSV" ຢູ່ແຖບດ້ານເທິງ ເພື່ອເລືອກໄຟລ໌ຂໍ້ມູນທີ່ທ່ານ Export ອອກມາຈາກ AppSheet. ລະບົບຈະທຳການອ່ານ ແລະ ສ້າງຕາຕະລາງສະຫຼຸບຜົນໃຫ້ອັດຕະໂນມັດ.

2. ການກັ່ນຕອງຂໍ້ມູນ (Filters)

  • ສາມາດກັ່ນຕອງໄດ້ຫຼາຍເງື່ອນໄຂ (Host, Domain, Type, Level, Date, Participant Type).
  • ສາມາດເລືອກໄດ້ຫຼາຍອັນພ້ອມກັນ (Multi-select) ໃນແຕ່ລະໝວດໝູ່.
  • ຫຼັງຈາກເລືອກແລ້ວ ໃຫ້ກົດປຸ່ມ "ຕົກລົງ (ປິດ)" ຢູ່ລຸ່ມສຸດຂອງໜ້າຕ່າງ Filter ນັ້ນ.
  • ກົດປຸ່ມ "ລ້າງການກອງທັງໝົດ" ຢູ່ມຸມຂວາເທິງຂອງແຜງກັ່ນຕອງ ເພື່ອຍົກເລີກເງື່ອນໄຂທັງໝົດ.

3. ຮູບແບບການຄິດໄລ່ (Count Type)

  • Count (ນັບທັງໝົດ): ລະບົບຈະນັບທຸກໆແຖວຂໍ້ມູນ (ເໝາະສຳລັບການນັບຈຳນວນຄັ້ງການເຂົ້າຮ່ວມ).
  • Distinct (ນັບບໍ່ຊ້ຳ ID): ລະບົບຈະນັບສະເພາະ Participant ID ທີ່ບໍ່ຊ້ຳກັນ (ປ້ອງກັນການນັບບຸກຄົນດຽວກັນຊ້ຳຊ້ອນກັນ).

4. ການເຊື່ອງຖັນ ແລະ ເຊື່ອງແຖວ

  • ເຊື່ອງຖັນ (Hide Column): ກົດປຸ່ມຮູບຕາປິດ 👁️ ຢູ່ເທິງຫົວຂໍ້ຖັນທີ່ທ່ານບໍ່ຕ້ອງການໃຫ້ສະແດງໃນລາຍງານ.
  • ເຊື່ອງແຖວ (Hide Row): ເອົາເມົ້າໄປຊີ້ໃສ່ຊື່ Event Name ໃນຕາຕະລາງ, ຈະມີໄອຄອນຕາປະກົດຂຶ້ນ. ເມື່ອກົດເຊື່ອງ ແຖວນັ້ນຈະຖືກເຊື່ອງ ແລະ ຕົວເລກຈະຖືກຕັດອອກຈາກການຄຳນວນຜົນລວມ (Grand Total).
  • ສາມາດກົດປຸ່ມ "ສະແດງທຸກຖັນ" ຫຼື "ສະແດງທຸກແຖວ" ທີ່ຢູ່ຂ້າງປຸ່ມ Export ເພື່ອຮຽກຂໍ້ມູນທີ່ເຊື່ອງໄວ້ກັບຄືນມາໄດ້ສະເໝີ.

5. ການສົ່ງອອກ (Export Excel)

ກົດປຸ່ມ "ສົ່ງອອກລາຍງານ Excel" ເພື່ອດາວໂຫຼດໄຟລ໌ .xlsx. ໄຟລ໌ Excel ຈະສະແດງສະເພາະຂໍ້ມູນທີ່ທ່ານກັ່ນຕອງ ແລະ ບໍ່ນຳເອົາຖັນ ຫຼື ແຖວທີ່ທ່ານເຊື່ອງໄວ້ອອກໄປນຳ, ພ້ອມທັງມີການສະແດງເງື່ອນໄຂ Filter ທີ່ໃຊ້ທັງໝົດໄວ້ເທິງຫົວຕາຕະລາງ.
